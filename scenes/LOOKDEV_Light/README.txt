The Lookdev_light is a light setup that must be used by everyone texturing the model.
This will contain a simple HDRI Skydome light.
The light should be a NEUTRAL light, that means no colour.
Props should never be textured with coloured lights.


Here is a great lookdev kit that you can download.  Determine one light that you will use for first pass:
